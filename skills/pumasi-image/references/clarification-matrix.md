## 카테고리 풀

### 스타일 (visual_style)

| 선택지 | 설명 |
|--------|------|
| 사실적 사진 | 실사 느낌, 구체적 디테일, 카메라 렌더링 |
| 3D 렌더링 | 게임·제품 시각화 느낌, 볼륨감 |
| 2D 일러스트 | 벡터 + 평면 채색, 깔끔함 |
| 수채화/손그림 | 따뜻하고 인간적인 느낌 |
| 미니멀 플랫 | 단순한 도형, 여백 중심 |
| 사이버 와이어프레임 (★창의) | 청사진 격자선, 기술적 느낌 |
| 자동 추천 | 내용에 맞게 AI 판단 |

|--------|------|

| 따뜻함 | 친근, 햇살, 파스텔 |
| 다크 시네마틱 | 영화적, 깊은 그림자 |
| 미니멀 클린 | 여백, 단순함, 모던 |
| 장엄함 | 스케일 큰, 서사적 |
| 조용한 위로감 (★창의) | 잔잔함, 여운 |
| 자동 추천 | AI 판단 |

### 색감 (color_palette)

|--------|------|

| 듀오톤 | 두 가지 색 대비 |

## 딸깍 원칙 적용

- 각 카테고리 **5개 이상 선택지**
- **1~2개는 창의적 대안** (★ 표시) — 예상 밖 선택으로 아이디어 확장
- **"자동 추천" 안전망** — 결정 피로 시 Claude 위임
- 입력에서 이미 명시된 카테고리는 해당 슬롯 스킵 → 다음 우선순위 카테고리로 대체

---

## 조건부 슬롯 대체 규칙

사용자 입력에 이미 명시된 카테고리가 있으면, 해당 슬롯은 다음 우선순위로 대체:

1. 모드 매트릭스 기본 3개 → 확정된 것 스킵
2. 스킵된 슬롯은 모드별 보조 카테고리로 대체:
   - 썸네일: 용도→스타일→텍스트공간 외에 분위기, 색감 순
   - 인물: 스타일→분위기→구도 외에 색감, 의상 순
   - 로고: 스타일→색감→사용맥락 외에 분위기, 브랜드톤 순
   - 풍경: 분위기→색감→구도 외에 시간대, 스타일 순
   - 일러스트: 스타일→분위기→색감 외에 구도, 주제 순
   - 제품: 스타일→배경→구도 외에 색감, 각도 순
   - 컨셉트: 스타일→분위기→색감 외에 구도, 상징 순

최종 질문 수: **0~5개** (일반 3~4개).
